ABSORB:

Absorb - это ключевое слово способности предотвращения повреждений, записанное как Absorb N, где N - любое натуральное число. Он предотвращает N повреждений от каждого отдельного источника, который наносит урон существу. Absorb был представлена ​​в Future Sight. По своей природе как способность предотвращения повреждений это белая способность. Пока что он напечатан только на одной карточке,Timeshifted Lymph Sliver. Эта механика была вдохновлена ​​механикой, созданной Wizards of the Coast для Коллекционной карточной игры «Звездных войн» под названием «Броня». В итоге он оказался слишком мощным и слишком сильно заглушил игру, чтобы ее можно было повторно использовать в большем количестве.
From the Comprehensive Rules (September 24, 2021—Innistrad: Midnight Hunt) 702.64. Absorb 702.64a Absorb is a static ability. “Absorb N” means “If a source would deal damage to this creature, prevent N of that damage.” 702.64b Each absorb ability can prevent only N damage from any one source at any one time. It will apply separately to damage from other sources, or to damage dealt by the same source at a different time. 702.64c If an object has multiple instances of absorb, each applies separately.
